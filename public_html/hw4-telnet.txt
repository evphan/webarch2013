II. Explain what happens when your browser loads http://people.ischool.berkeley.edu/~<USER>/server/wiki from sending the request to loading HTML
	Answer: The browser sends an http request to get the page. When the request is accepted by the web server, the web server responded with a 301 code for moved permanently. Then the broswer connects to the web server at http://www.ischool.berkeley.edu/ and asks for the new page: http://www.ischool.berkeley.edu/evphan/server/wiki. The webserver then responds with a 404 page not found.
III. What is the return code for http://people.ischool.berkeley.edu/~<USER>/server/?
	Answer: 301 Moved Permanently
